Student Management System

This project is a simple Student Management System built using PHP and MySQL.
The system allows users to create an account, log in, and manage student records (add, edit, delete, and view).

Features

User registration (Sign Up)

User login and logout

Protected pages using sessions

Add new students

View students list

Edit student information

Delete students

Each user can manage only their own students

Requirements

PHP

MySQL

Web server (XAMPP / Laragon)

How to Run

Create a database called:

student_management


Import the SQL file (if provided) or run the system and tables will be created automatically.

Update database connection in db.php if needed.

Open the project in browser:

http://localhost/student-management-system/


Notes

The system uses sessions to protect pages.

Passwords are stored securely using hashing.

Prepared statements are used to protect against SQL injection.
